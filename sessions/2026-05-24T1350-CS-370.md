## Side-findings (noted; deferred)

These were discovered during the session but explicitly scope-locked OUT of this ship per `feedback_strict_plan_execution`:

1. **`lobes/doc-writer-lobe/test-fixtures/api-node-minimal/input/DocResult.json` carries a top-level `_description` field** that the schema rejects under `additionalProperties: false`. Convention is widespread (7 hits across test-writer-lobe + doc-writer-lobe input fixtures: `grep -l '"_description"' lobes/*/test-fixtures/**/input/*.json` → 7 files). Two interpretations: (a) authors intended `_description` as documentation-on-fixture (the underscore-prefix convention) but the schemas were authored with strict `additionalProperties: false` from day one; (b) schemas should accept underscore-prefixed metadata fields. Resolution belongs in a fixture-hygiene follow-on; not patched here. Workaround: smoke against a `jq 'del(._description)'`-stripped copy for happy-path exit-0 verification.
2. **`agents/global/migration-writer.md` (1629 bytes, dated 19 Mar)** lingers post-ADR-037 retirement — `feedback_deploy_sh_additive_no_prune` predicts this exact class. CS-368 ship did not include an explicit `rm` step for this stale legacy free-form agent. Cleanup is a 1-line patch; deferred.
3. **CS-103 (Pre-Release Regression Suite — Playwright)** carries the orphan-Playwright vision from CS-294's original scope. Its body still references Phase 1/2/3 infrastructure pre-retirement; post-ADR-037 the framing is stale (Regression Lobe gone; only Playwright-as-runtime-service concern survives). Refresh deferred.
4. **No `chains/design-driven-build.json`** referenced by orchestration-architecture.md:37 exists on disk. Trivial stale wikilink class; the orchestration-architecture doc has aspirational sketches that haven't materialized as chain files. Acceptable as documentation-of-direction; not patched.
