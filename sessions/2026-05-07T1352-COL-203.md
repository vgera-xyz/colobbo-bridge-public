## Methodology notes (carry-forward for future sessions)

- **canonical_pages is array-of-objects, not a map.** Format: `{path, topic_keywords}`. Append via `jq '.wiki.canonical_pages += [{...}, {...}]'` not `+= {key: value}`. Update memory accordingly.
- **Frontmatter status enum: [active, stub, superseded, archived]** — `frozen` not allowed despite semantic fit. Use `status: active` + body or separate frontmatter field for frozen-baseline marker.
- **Colobbo team has no "Done" state** — terminal completed-type is "Released". Map "Done" → "Released" colloquially.
- **Set-A − set-B = "unmapped" ≠ "dead-code".** Five distinct possibilities (truly dead, cross-module, trace-skipped, newer-than-trace, summary-only). Bias REVIEW with evidence-tag default; reserve DROP for multi-signal corroboration.
- **Audit-trace route normalization:** audit uses `api/X/...`, traces use `/api/X/...`. Strip leading slash from both sides before set comparison.
