## Summary

CS-297 closes the silent-drift gap that surfaced **53 fixtures across 6 Lobes** when the Lobe Regression Suite was manually dispatched after [[2026-05-12T1528-CS-142|CS-142]] (run [25747521407](https://github.com/vgera-xyz/colobbo-agent-system/actions/runs/25747521407)). Root cause: 3 ships across 5 cumulative commits (CS-141 handoff `bfe3597`+`b6e4b9c`, CS-281 `8c998ee`, CS-142 `170f0dc`+`d9b4b23`) added fintech-glaze edits without paired sibling-Lobe re-snap. Drift was 100% additive — `glaze._version` byte-identical at 1.13.0 in both drifted snapshots and live; only new fields added.

Three-layer structural fix (mirrors Pattern K "humans forget" precedent):

1. **Detection** — new invariant **(w)** in `scripts/verify-glaze-shape.sh` flags drift at pre-commit time. Extends Gate 1; runs on every PR + every local push. Capability-gated via `synthetic_glaze: true` flag per [[../../decisions/adr-009-multi-dialect-tooling-reality|ADR-009]].
2. **Remediation** — new `scripts/resnap-additive-fixtures.sh`. One-command idempotent re-snap with capability-gating idiom matching invariant (w) byte-for-byte. USE ONLY for additive-only drift (operator-discipline-enforced).
3. **Discipline** — new **Pattern Q** in [[../../raku-meta/state-drift-patterns|state-drift-patterns.md]] codifies the rule + worked example. Canonical reference for CS-143 / CS-144 / CS-145 Phase-3b-equivalent ship-procedure step.

No glaze edit this ship. Pure script + invariant + docs. Glaze `_version` stays 1.13.0. Fixture inventory unchanged (58 total, 53 non-synthetic, 5 synthetic: 4 database-lobe + 1 mobile-lobe).

## Symmetric scrutiny — detection vs remediation parity

The detection invariant (w) function in `verify-glaze-shape.sh` and the remediation loop in `resnap-additive-fixtures.sh` MUST stay in lockstep on capability-gating logic, or drift detection and re-snap remediation diverge (a synthetic case might be flagged drifted by detection but skipped by remediation, or vice-versa). Verified at handoff time:

- **Iteration glob:** both use `lobes/*-lobe/test-fixtures/*/glaze-snapshot.json` ✓
- **Synthetic lookup jq idiom:** both use `.cases[]? | select(.id==$id) | .synthetic_glaze // false` with `--arg id "$case_id"` ✓
- **Fallback chain:** both use `2>/dev/null || echo "false"` + `[ -z "$synthetic" ] && synthetic="false"` belt-and-suspenders ✓
- **Drift comparison:** both use `diff -q` exit-status (detection negates for fail, remediation negates for skip) ✓

If a future change touches one, the other MUST be updated lockstep; Pattern Q text codifies this.
