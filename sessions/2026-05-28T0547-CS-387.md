Severity heuristic (hardcoded at v0.1.0; Ware-agnostic enum mapping per script header docblock):
- **HIGH**: stage `lobe_status` ∈ {error, escalated, escalated_to_architect, aborted_payload_overflow}; run-level escalation block non-null; advisory `.type === "pending_human_review"`; ANY advisory whose `.message` contains substring `high_risk_ownership` (case-insensitive — the canonical motivating case).
- **MEDIUM**: advisory `.type` ∈ {scope_caveat, tradeoff, framework_mismatch, module_unresolvable, tooling_value_unknown, non_backend_components_only, target_repo_unresolvable, ambiguous_classification}; codegen skip reason ∈ {target_repo_unresolvable, persistence_idiom_unmapped, source_path_unresolvable, arch_component_unresolvable}; UNKNOWN advisory `.type` → MEDIUM (surface, don't bury — fail-loud-not-silent).
- **LOW**: advisory `.type` ∈ {yagni_flag, installed_package_unavailable, capability_not_enabled}; codegen skip reason ∈ {yagni_flag, installed_package_unavailable, brownfield_modify_deferred, output_exists, duplicate_component, unsupported_stack, persistence_idiom_unresolved}.
