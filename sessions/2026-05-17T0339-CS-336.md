## Scout deviations (STOP-gate, operator-resolved)

1. **`/cibo-handoff` is `SKILL.md` prose, LLM-executed — not code.** No retry library, no shell-invocable MCP call. The prompt's "wrap the MCP call in a retry library / mocked-502 regression test" is structurally inapplicable. → B1 retry is a written **prose contract**; the regression test is the achievable mechanical part (deliberate-break, ADR-015 D2 shape).
2. **Bug-A correct shape is β (permanent `status: operational` queue), not per-write `staged-deferred`.** ADR-015 D1 + the *working* sibling `handoff-pending-comments.md` (Step 3's fallback target — never breaks Gate 16 because it's pre-seeded + append-only). `staged-deferred` (the ticket/58de6e4b shape) was correct for that one-shot retrofit but wrong systemically (also trips the 14-day WARN). `feedback_artifact_rules_over_ticket_prose`.
